Buttons allow users to take actions, and make choices, with a single tap.They have multiple styles for various needs, and are ideal for calling attention to where a user needs to do something in order to move forward in a flow.

```
<ButtonGroup variant="secondary" splitButton>
  <Button>I'm a button</Button>
  <Button>I'm a button</Button>
</ButtonGroup>
```

```jsx{live:false,preview:false}
<Button
  icon={
    <Icon>
      <path d="M14 7h-5v-5h-2v5h-5v2h5v5h2v-5h5v-2z" />
    </Icon>
  }
>
  I'm a button
</Button>
```
